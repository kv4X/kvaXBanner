# kvaXBanner
Teamspeak3 Banner info for samp&ts3 servers.

<b>Installation</b><br>
1. PHP-GD(GD Graphics Library).<br>
Ubuntu 14.04 (sudo apt-get install php5-gd)<br>
2. Set banner update interval to 60 seconds.<br>
